Perjalanan kita di dunia *testing* telah membawa kita dari fondasi kode (*Unit Testing*) dan "dapur" sistem (*API Testing*). Sekarang, materi dari Kelompok 7 membawa kita ke puncak piramida: **Automation Testing** untuk *End-to-End (E2E)*. Dan di dunia automasi *browser*, ada satu nama yang merajai: **Selenium**.

Ini pada dasarnya adalah cara kita memprogram "robot" untuk melakukan semua pekerjaan manual yang membosankan: membuka *browser*, mengisi formulir, mengklik tombol, dan memvalidasi hasilnya.

### Apa Itu Selenium & Selenium WebDriver?

    <Text variant="body-default-s" onBackground="neutral-weak">
      Ini adalah **framework open-source** untuk mengotomatisasi *browser*. Selenium memungkinkan kita menulis skrip (dalam bahasa seperti Python, Java, C#) yang bisa mengontrol *browser* (Chrome, Firefox, Safari, Edge) untuk menguji aplikasi web secara otomatis.

    <Text variant="body-default-s" onBackground="neutral-weak">
      Ini adalah **komponen inti** dari Selenium. Anggap saja WebDriver sebagai "jembatan" atau "penerjemah". Skrip kita (misal: "klik tombol login") memberi perintah ke WebDriver, lalu WebDriver-lah yang menerjemahkan perintah itu menjadi instruksi yang dipahami oleh masing-masing *browser*.

</Row>

Materi Kelompok 7 memberikan contoh *real-world* menggunakan *website* demo `saucedemo.com`.

**2. Kode Automasi (Contoh TC-001 dalam Python)**
Sekarang, inilah "keajaiban" Selenium. Kita menerjemahkan `TC-001` di atas menjadi kode yang bisa dieksekusi:

<CodeBlock
    marginBottom="24"
    codes={[
      {
        code:
`from selenium import webdriver
from selenium.webdriver.common.by import By
import time

# 1. Arrange: Siapkan driver dan buka browser
driver = webdriver.Chrome()
driver.get("https://www.saucedemo.com")

# 2. Act: Lakukan langkah-langkah login
driver.find_element(By.ID, "user-name").send_keys("standard_user")
driver.find_element(By.ID, "password").send_keys("secret_sauce")
driver.find_element(By.ID, "login-button").click()

# Jeda singkat untuk melihat halaman dimuat
time.sleep(2)

# 3. Assert: Validasi hasilnya
expected_url = "https://www.saucedemo.com/inventory.html"
actual_url = driver.current_url

assert actual_url == expected_url, f"Login gagal! URL aktual: {actual_url}"

print("Test TC-001: Login Sukses... LULUS!")

# Cleanup
driver.quit()
`,
        language: "python",
        label: "Contoh Skrip: test_login_sukses.py"
      }
    ]}
